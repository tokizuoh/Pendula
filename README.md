# Pendula
Repository for verification related to iOS apps.
  
## Xcode
  
```bash
> xcodebuild -version
Xcode 12.3
Build version 12C33
```
  
## Setup
  
```
> brew install mint
> mint --version
Version: 0.14.2
```
  
```
> git clone git@github.com:tokizuoh/Pendula.git
> cd Pendula
> mint bootstrap  
> xcodegen
> xed .
```
